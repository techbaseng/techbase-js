About This Course

Make the web
come alive.

HTML gives pages structure, CSS makes them beautiful — JavaScript makes them interactive. It's the programming language of the browser, and the most widely used language in the world.

46 lessons across 5 phases, covering everything from syntax to advanced patterns, real projects, and a complete reference manual. Designed by Babatunde Awoyemi, Techbase, Ibadan.

  • Functions, closures, classes and modules
  • Selecting and manipulating the DOM
  • Events, forms and user interaction
  • Async programming — promises, async/await, Fetch API
  • JSON, AJAX, Web APIs, Canvas and jQuery
